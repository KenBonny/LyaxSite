Cookie Policy

I strongly believe cookies are food!

This is why my website only uses one functional cookie.

The cookie: __cfduid

This is a Cloudflare cookie that we need to have the website up and running.

What does the cookie do?

Cloudflare’s full explanation can be found here.

It’s a temporary cookie that expires after 30 days. The cookie collects an anonymizes the IP addresses of visitors to detect if there is any malicious intent.

The cookie does not:

  • Allow for cross-site tracking
  • Follow users from site to site by merging various _cfduid identifiers into a profile
  • Correspond to any user IB in a customer’s web application

The cookie is saved for less than 24h on Cloudflare server, as per agreement. If an IP address is flagged because of a security alert, there may be exceptions.
